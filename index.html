This project is about the Wordpress plugin for Shared-house.com booking calendar.

The plugin allows to easily display the booking calendar on a Wordpress site. The plugin homepage and dwnload information can be found here: http://www.shared-house.com/free-booking-calendar.php

However a number of improvements need to be made, for instance:

  • The ability to link to an internal page of the wordpress site instead of the calendar page on shared-house.com
  • the ability to include several calendars with different ad numbers
